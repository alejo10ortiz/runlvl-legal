## 2. Descripción del servicio

RUNLVL es una aplicación de entrenamiento de running que:
- Genera planes de entrenamiento personalizados para distancias de 5K, 10K, 21K y 42K.
- Registra sesiones de entrenamiento con métricas GPS (distancia, ritmo, ruta).
- Ofrece un sistema de progresión gamificada (XP, niveles y logros).
- Muestra información meteorológica contextual para cada sesión.
